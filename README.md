## Features

- **Animated Hero Background** — The brand logo floats and pulses across the hero section using an HTML5 Canvas animation with mouse parallax
- **Brand-Accurate Design** — Royal navy blue and gold colour palette matched directly from the official logo
- **Services Menu** — 6 services displayed with descriptions and ZAR pricing
- **Online Booking Form** — Name, phone, service selection, date and time picker with confirmation message
- **About Section** — Business story, stats (6+ years, 4 barbers, 5k+ clients)
- **Gallery Section** — 8-cell photo grid (replace placeholders with real photos)
- **Testimonials** — 3 client reviews
- **Contact Info** — Address, phone, WhatsApp, email, and opening hours
- **Smooth Scroll Navigation** — Fixed navbar with anchor links
- **Responsive Layout** — Works on desktop and mobile browsers

## How to Use

### Run Locally
```bash
# Just open the file — no install needed
open index.html
```

### Deploy to GitHub Pages
```bash
git init
git add index.html README.md
git commit -m "Initial commit — Msimang & Co. website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/msimang-barbershop.git
git push -u origin main
```
Then go to **Settings → Pages → Source → main branch** and your site will be live at:
`https://YOUR_USERNAME.github.io/msimang-barbershop`

## Project Pitch — How This Website Helps the Business Grow

### Problem It Solves
Most local barbershops in Durban rely on walk-ins and word of mouth. Msimang & Co. had no digital presence, meaning potential clients searching online couldn't find them.

### What the Website Does
1. **Captures online traffic** — shows up in Google searches for "barbershop Durban" or "fade haircut Berea"
2. **Converts visitors to bookings** — the booking form removes friction; clients don't have to call
3. **Builds trust before they walk in** — services, pricing, testimonials, and the professional brand identity all signal quality
4. **Works 24/7** — clients can book at midnight, on a Sunday, without the barber lifting a finger

### Business Impact
- Estimated **+30% increase in bookings** by capturing clients who would otherwise go elsewhere
- Reduces no-shows with booking confirmation and WhatsApp follow-up
- Professional brand presence to compete with larger salons
